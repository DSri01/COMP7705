# Image details

- Metadata for one slot in the image **chain** (chain index, ids, upload and scan times, file size).
- **File** and **scan** badges match the component page (awaiting upload → ready; scan OK / scanning / not uploaded).
- **Upload .tar** — shown when file status is awaiting upload.
- **Current image** — link back to the component page for scan and image-CVE triage.
- **Historical image** — older chain slots are read-only here; triage applies only to the **current** image on the component page.
