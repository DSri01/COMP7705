# CVE catalog

- Global list of CVEs known to the platform.
- **Open CVE** — jump to a CVE by id without browsing the table.
- **Create CVE** — register a new canonical id (e.g. `CVE-2021-44228`).
- Table columns: **CVE**, **severity**, **Summary** (intel snippet; hover for last intel update), **View**.
- Use **previous** / **next** paging when more than one page of results.
- **View** opens full intel highlights, research summary, and documents on the CVE detail page.
- Intel refreshes over time; use **Update intel** on the CVE page to refresh one CVE now.
