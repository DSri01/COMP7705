# Project detail

- **Project profile** — name and description (maximize/minimize); **Update Project** edits the description only.
- **Stats charts** — project-wide CVE breakdown by decision and severity.
- **Components** — list of deployable units in this project.
- **Create Component** — add a new component.
- Each component card shows name, id, last updated, per-component **stats charts**, and **Inspect Component** (images and image-CVE triage).
