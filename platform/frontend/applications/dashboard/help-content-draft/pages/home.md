# Getting started

## What this platform does

- Manage **projects**, **components**, and **container images**.
- Track **CVEs** per image with decisions and advice.
- Enrich CVEs with **NVD, EPSS, and CISA KEV** intel.
- Use the **Platform Agent** for research, triage, and guidance.

## Navigation

- Top bar: **Home**, **Projects**, **CVEs**, **Platform Agent** — same destinations as the cards below.

## Suggested study workflow

1. **Projects** → open a **component**.
2. **New image** → upload `.tar` → **Scan current image**.
3. Review **image-CVEs** on the component page.
4. Open a **CVE** → check intel and research documents.
5. Try the **Platform Agent** — name the **CVE id** if you want web pages saved to research documents.
