# Add CVEs

Link CVEs to the **current image**.

## Add single CVE

- Enter a canonical id (e.g. `CVE-2021-44228`) and **Create + link**.
- Missing ids are registered in the catalog first, then linked.

## Import scanner JSON

- Paste or upload JSON: `{"VulnerabilityIDs":["CVE-..."]}`.
- **Import + link** registers any missing ids, then links all to the current image.
